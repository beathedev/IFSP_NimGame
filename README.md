# IFSP_NimGame
Desenvolvimento de jogo utilizando exclusivamente estruturas de repetição (for e/ou while), estruturas sequenciais e condicionais e o comando randomize, sem o uso de vetores, matrizes, funções ou procedimentos

O Jogo do Nim é um clássico jogo matemático de estratégia com origem milenar. A implementação desenvolvida recria a versão mais popular: a dos 21 palitos.

A ideia central é simples e envolvente: dois jogadores (neste caso, um humano e o computador) se alternam retirando palitos de uma mesa. A tensão do jogo vem da regra que inverte a lógica comum — em vez de querer pegar o máximo possível, o objetivo é forçar o adversário a pegar o último palito.

Objetivo: Não ser o jogador que retirar o último palito da mesa.
